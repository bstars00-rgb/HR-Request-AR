## 🇰🇷 한국어

### 1. 접속
- 브라우저에서 **https://bstars00-rgb.github.io/HR-Request-AR/** 접속. 설치·회원가입 없음.
- PC·모바일 모두 사용 가능. 모든 구성원이 **같은 데이터를 실시간**으로 봅니다.

### 2. 언어 / 화면 모드
- 왼쪽 아래 **`EN / 한`** 버튼으로 한국어↔영어 전환.
- **달/해 아이콘**으로 다크/라이트 모드 전환.

### 3. 일정 유형 (색상)
| 색 | 유형 | 예시 |
|---|------|------|
| 🟢 초록 | **박람회/세일즈콜** | Thai Travel Fair, China Sales Call |
| 🟡 노랑 | **출장** | 출장(심천), 자카르타 출장 |
| ⚪ 회색 | **내부업무/OKR** | OKR 미팅, 대시보드 고도화 |
| 🟣 보라 | **개인/휴가** | 연차, 개인 일정 |
| 🔵 남색 | **기타** | 그 외 |

### 4. 화면 소개
| 메뉴 | 내용 |
|------|------|
| **대시보드** | 오늘/이번주/이번달 일정, 오늘 자리 비움(출장·개인), 팀별·유형별 집계 |
| **캘린더** | 월간 달력에서 일정 보기 + **날짜 클릭으로 등록, 일정 클릭으로 수정/삭제**. 팀·구성원·유형 필터, 인쇄·내보내기 |
| **구성원** | 팀 멤버 명단. **내보내기(CSV)** |
| **팀 현황** | 팀별 인원·이번 주 일정·예정 일정 |
| **설정** | (관리자) 팀 관리, 공휴일, 편집 잠금 PIN |

### 5. 자주 하는 일 (전부 캘린더에서)

**일정 등록하기**
1. **캘린더**에서 **원하는 날짜 칸을 클릭** (또는 우측 상단 **+ 일정 등록**)
2. 구성원(본인) / 유형 / 기간 / 내용 입력 → **일정 등록**
3. 즉시 모두에게 공유됩니다.

**일정 수정/삭제**: 달력의 **일정(색깔 막대)을 클릭** → 수정 또는 삭제.

**내 일정만 보기**: 캘린더 상단 **구성원 필터**에서 본인 이름 선택.

**엑셀로 내보내기**: 캘린더 우측 상단 **내보내기(CSV)** → 표시 중인 달의 일정 다운로드.

**캘린더 인쇄**: 캘린더 우측 상단 **인쇄** → 달력만 출력.

### 6. 관리자용
- **편집 잠금**: 설정에서 **PIN**을 정하면 편집 시 왼쪽 아래 **잠금 해제**로 PIN 입력 필요. 비워두면 누구나 편집.
- **구성원 추가/수정**: 구성원 → **+ 직원 추가** (이름·팀·직급·입사일).
- **팀 관리**: 설정 → 팀 추가/삭제·매니저.
- **공휴일**: 설정 → 국가별 공휴일 등록 (캘린더에 표시됨).

### 7. 자주 묻는 질문
- **다른 사람도 같은 데이터를 보나요?** 네. 실시간으로 같은 데이터를 봅니다.
- **저장 버튼이 없어요.** 입력하면 자동 저장됩니다.
- **수정 버튼이 안 보여요.** 편집 잠금 상태입니다. 왼쪽 아래 **잠금 해제**로 PIN 입력.

## 🇬🇧 English

### 1. Access
- Open **https://bstars00-rgb.github.io/HR-Request-AR/** in a browser. No install, no sign-up.
- Works on PC and mobile. Everyone sees the **same data in real time**.

### 2. Language / Theme
- Use the **`EN / 한`** button (bottom-left) to switch English↔Korean.
- Use the **sun/moon icon** for dark/light mode.

### 3. Schedule categories (colors)
| Color | Category | Examples |
|---|------|------|
| 🟢 Green | **Fair / Sales Call** | Thai Travel Fair, China Sales Call |
| 🟡 Yellow | **Business Trip** | Trip (Shenzhen), Jakarta trip |
| ⚪ Gray | **Internal / OKR** | OKR meeting, dashboard work |
| 🟣 Purple | **Personal / Leave** | Annual leave, personal |
| 🔵 Indigo | **Other** | Anything else |

### 4. Screens
| Menu | What it does |
|------|------|
| **Dashboard** | Today / this week / this month, who's away today (trip·personal), counts by team & category |
| **Calendar** | Monthly calendar — **click a day to add, click an entry to edit/delete**. Team/member/category filters, print & export |
| **Members** | Team member list. **Export (CSV)** |
| **Teams** | Headcount, this week's entries, upcoming per team |
| **Settings** | (Admin) teams, holidays, edit-lock PIN |

### 5. Common tasks (all in the Calendar)

**Add a schedule entry**
1. On the **Calendar**, **click the day** you want (or **+ Add Schedule**, top right)
2. Fill in member (you) / category / dates / details → **Add Schedule**
3. Shared with everyone instantly.

**Edit / delete an entry**: **click the colored entry** on the calendar → edit or delete.

**See only my schedule**: pick your name in the **member filter** above the calendar.

**Export to Excel**: **Export (CSV)** (top-right of Calendar) → downloads the displayed month.

**Print calendar**: **Print** (top-right of Calendar) → prints just the calendar.

### 6. For Admins
- **Edit lock**: Set a **PIN** in Settings; editing then requires **Unlock** (bottom-left). Empty PIN = anyone can edit.
- **Add/edit members**: Members → **+ Add Employee** (name, team, position, join date).
- **Teams**: Settings → add/remove teams, managers.
- **Holidays**: Settings → register holidays by country (shown on the calendar).

### 7. FAQ
- **Does everyone see the same data?** Yes — shared in real time.
- **There's no Save button.** Changes save automatically.
- **I don't see edit buttons.** Editing is locked. Click **Unlock** (bottom-left) and enter the PIN.

## 🇻🇳 Tiếng Việt

> Giao diện có tiếng Hàn/tiếng Anh. Hướng dẫn này dùng nhãn **tiếng Anh** (bấm **`EN`** ở góc dưới bên trái).

### 1. Truy cập
- Mở **https://bstars00-rgb.github.io/HR-Request-AR/** trên trình duyệt. Không cần cài đặt hay đăng ký.
- Dùng được trên máy tính và điện thoại. Mọi người xem **cùng một dữ liệu theo thời gian thực**.

### 2. Ngôn ngữ / Giao diện
- Nút **`EN / 한`** (góc dưới trái) để đổi Anh↔Hàn.
- Biểu tượng **mặt trời/mặt trăng** để đổi nền sáng/tối.

### 3. Loại lịch (màu sắc)
| Màu | Loại | Ví dụ |
|---|------|------|
| 🟢 Xanh lá | **Fair / Sales Call** (hội chợ / sales) | Thai Travel Fair, China Sales Call |
| 🟡 Vàng | **Business Trip** (công tác) | Công tác Thâm Quyến, Jakarta |
| ⚪ Xám | **Internal / OKR** (việc nội bộ) | Họp OKR, làm dashboard |
| 🟣 Tím | **Personal / Leave** (cá nhân / nghỉ) | Nghỉ phép, việc cá nhân |
| 🔵 Chàm | **Other** (khác) | Việc khác |

### 4. Các màn hình
| Menu | Chức năng |
|------|------|
| **Dashboard** | Lịch hôm nay / tuần này / tháng này, ai vắng mặt hôm nay, thống kê theo nhóm & loại |
| **Calendar** | Lịch tháng — **bấm vào ngày để thêm, bấm vào lịch để sửa/xóa**. Lọc theo nhóm/thành viên/loại, in & xuất file |
| **Members** | Danh sách thành viên. **Export (CSV)** |
| **Teams** | Số người, lịch tuần này, lịch sắp tới theo nhóm |
| **Settings** | (Quản trị) nhóm, ngày lễ, PIN khóa chỉnh sửa |

### 5. Thao tác thường dùng (tất cả trên Calendar)

**Đăng ký lịch**
1. Trên **Calendar**, **bấm vào ngày** bạn muốn (hoặc **+ Add Schedule**, góc trên phải)
2. Nhập thành viên (bạn) / loại / thời gian / nội dung → **Add Schedule**
3. Chia sẻ ngay với cả nhóm.

**Sửa / xóa lịch**: **bấm vào ô lịch có màu** trên Calendar → sửa hoặc xóa.

**Chỉ xem lịch của tôi**: chọn tên bạn ở **bộ lọc thành viên** phía trên Calendar.

**Xuất ra Excel**: nút **Export (CSV)** (góc trên phải Calendar) → tải lịch của tháng đang xem.

**In lịch**: nút **Print** (góc trên phải Calendar) → chỉ in phần lịch.

### 6. Dành cho Quản trị
- **Khóa chỉnh sửa**: Đặt **PIN** trong Settings; muốn sửa phải bấm **Unlock** (góc dưới trái). PIN trống = ai cũng sửa được.
- **Thêm/sửa thành viên**: Members → **+ Add Employee** (tên, nhóm, chức vụ, ngày vào).
- **Nhóm**: Settings → thêm/xóa nhóm, quản lý.
- **Ngày lễ**: Settings → thêm ngày lễ theo quốc gia (hiển thị trên lịch).

### 7. Câu hỏi thường gặp
- **Mọi người có xem cùng dữ liệu không?** Có — theo thời gian thực.
- **Không có nút Lưu.** Thay đổi được lưu tự động.
- **Tôi không thấy nút chỉnh sửa.** Đang bị khóa. Bấm **Unlock** (góc dưới trái) và nhập PIN.
